# IELTS Momentum Engine

This is a clean, minimal, and student-friendly dashboard UI created for the IELTS preparation product "Leap Scholar". It is built with vanilla HTML, CSS, and Chart.js.

## Features

- **Diagnostic Onboarding**: Mock diagnostic test and goal setting.
- **Personalized Roadmap**: Visual 4-week timeline.
- **Daily Habit Loop**: Gamified "Daily Tasks" to build momentum.
- **Progress Analytics**: Band prediction and accuracy trends using interactive charts.

## Running Locally

To run this project locally, you can use any simple HTTP server.

Example with Python:
```bash
python -m http.server 8000
```
Then open `http://localhost:8000` in your browser.

## Deployment on Vercel

This project is ready for one-click deployment on Vercel.

1.  Push this repository to GitHub.
2.  Log in to [Vercel](https://vercel.com).
3.  Click "Add New Project" and import your repository.
4.  Vercel will automatically detect it as a static site.
5.  Click **Deploy**.

Alternatively, drag and drop the project folder onto the Vercel dashboard if you are not using GitHub.
